Schematica MCP Server Architecture

Model Context Protocol Server for Globomantics Robot Fleet Management

AI Assistant Layer Claude AI & MCP Configuration
MCP Server Components HTTP Server, Endpoints, Core Server
Transport Layer StreamableHTTPServerTransport
MCP Tools Robot Fleet Management Operations
API Integration Tool Handler & HTTP Client
Robot Fleet API Express.js Backend Server

Available Tools

  • list_robots List all robots with filters
  • get_robot Get specific robot details
  • create_robot Add new robot to fleet
  • update_robot Modify robot properties
  • delete_robot Decommission robot
  • schedule_maintenance Plan maintenance tasks

Server Configuration

  • MCP Port: 3001
  • Robot API URL: localhost:3000
  • Transport: Streamable HTTP
  • Protocol: MCP v1.0.0
  • SDK: @modelcontextprotocol/sdk

Request Flow

  • 1. Claude sends tool call request
  • 2. HTTP POST to /mcp endpoint
  • 3. Transport handles streaming
  • 4. Server routes to appropriate tool
  • 5. Handler calls Robot API
  • 6. Response streams back to Claude
